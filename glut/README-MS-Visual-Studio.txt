Where to put GLUT files in MS Visual Studio:

glut.h   -- C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\include\GL\
glut32.lib -- C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\lib\

On a 32 bit machine:
glut32.dll -- C:\Windows\System32\

On a 64 bit machine: 
glut32.dll -- C:\Windows\SysWOW64\
